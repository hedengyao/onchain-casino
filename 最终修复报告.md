## 🔨 修复的问题

### 1. Title 嵌套问题 ✅
**问题描述**: 二级页面显示一级页面的 title，导致标题重复

**根本原因**: 
- `layout.tsx` 使用了 `'use client'` 但同时又定义了 `metadata`
- 这在 Next.js 中是不兼容的

**修复方案**:
```tsx
// 修复前
'use client';  // ❌ 与 metadata 不兼容
import type { Metadata } from "next";
export const metadata: Metadata = { ... };

// 修复后
// 移除 'use client'，改为纯服务器组件
import type { Metadata } from "next";
export const metadata: Metadata = { ... };
```

**新建组件**:
- `src/components/Navbar.tsx` - 客户端组件，处理所有交互
- `src/components/Footer.tsx` - 客户端组件

### 2. 成就页面报错 ✅
**问题描述**: 点击成就功能会报错

**原因**: 成就页面不存在，但在首页有链接

**修复方案**:
- 移除了成就页面的链接
- 将成就功能整合到功能展示中，作为展示项

### 3. SSR 错误 ✅
**问题描述**: `window is not defined` 错误

**修复方案**:
```tsx
// 添加客户端检查
const [isClient, setIsClient] = useState(false);
useEffect(() => { setIsClient(true); }, []);

if (!isClient) {
  return <div className="loading" />;
}
```

### 4. 钱包连接功能 ✅
**问题**: 钱包连接组件有 SSR 问题

**修复**: 重构 `useWallet` hook，添加完整的客户端检查

### 5. 语言切换功能 ✅
**问题**: 语言切换组件独立存在，与导航栏分离

**修复**: 将语言切换整合到 Navbar 组件中
