## 🚀 步骤 2：部署到 Vercel

### 2.1 访问 Vercel

打开浏览器访问：[https://vercel.com/new](https://vercel.com/new)

### 2.2 登录/注册

- 使用 GitHub 账号登录（推荐）
- 或注册新账号

### 2.3 Import 项目

1. 点击 **Import Git Repository**
2. 在列表中找到 `hedengyao/onchain-casino`
3. 点击 **Import**

### 2.4 配置项目

**Project Name**: 自定义项目名称（可选）
**Framework Preset**: Next.js（自动识别）
**Root Directory**: `./`（默认）
**Build Command**: `pnpm build`（默认）
**Output Directory**: `.next`（默认）

### 2.5 添加环境变量（可选）

如果需要 OKX API 功能，点击 **Environment Variables** 添加：

| 变量名 | 值 |
|--------|-----|
| `OKX_API_KEY` | 你的 API 密钥 |
| `OKX_SECRET_KEY` | 你的密钥 |
| `OKX_PASSPHRASE` | 你的密码短语 |

> 获取 OKX API：[https://web3.okx.com/onchain-os/dev-portal](https://web3.okx.com/onchain-os/dev-portal)

### 2.6 点击 Deploy

点击 **Deploy** 按钮，等待 2-3 分钟

### 2.7 部署完成！

你会获得一个域名：
- `https://onchain-casino-xxx.vercel.app`

点击 **Visit** 访问你的应用

## 🔧 常见问题解决

### 问题 1：git push 失败 - 认证错误

**解决方案**：
1. 使用 Personal Access Token 代替密码
2. 访问 [GitHub Token 设置](https://github.com/settings/tokens)
3. 创建新 token（勾选 `repo` 权限）
4. 使用 token 作为密码推送

### 问题 2：Vercel 部署失败 - Build Error

**查看日志**：
1. 在 Vercel 项目页面点击 **Deployments**
2. 点击失败的部署
3. 查看 **Build Logs** 找到错误

**常见错误**：
- `Module not found`: 检查依赖是否安装
- `TypeScript error`: 检查类型定义

### 问题 3：钱包连接不工作

**原因**：浏览器未安装钱包插件

**解决方案**：
- MetaMask: [https://metamask.io/](https://metamask.io/)
- Phantom: [https://phantom.app/](https://phantom.app/)
