## 🎯 产品定位

**OnChain Casino** 是一个融合以下元素的社交交易赌场：

| 元素 | 来源 | 作用 |
|------|------|------|
| 🎲 **预测市场** | Polymarket | 对赌机制，二元结果预测 |
| 💹 **社交交易** | eToro | 跟单、带单、分润 |
| 🔥 **Meme 币交易** | DEX | 实时交易执行 |
| 🏆 **竞技排名** | 交易竞赛 | PVP 对战、大逃杀 |
| 🎮 **游戏化** | 成就系统 | NFT 徽章、排行榜 |

## 🚀 核心功能

### 1️⃣ 预测市场 (Polymarket 风格)

**玩法**：对代币价格走向进行二元预测

```
┌─────────────────────────────────────────────────────────┐
│  📊 SOL 24 小时内会突破 $200 吗？                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ✅ 会 (Yes)          ❌ 不会 (No)                       │
│  当前概率：65%        当前概率：35%                     │
│  1 份额 = $0.65       1 份额 = $0.35                    │
│  赔付：$1.00          赔付：$1.00                       │
│                                                         │
│  💰 奖池：$125,000    ⏰ 剩余：18 小时 32 分              │
│  👥 参与人数：2,847                                     │
│                                                         │
│  [ 下注 100 USDT ]                                      │
└─────────────────────────────────────────────────────────┘
```

**特点**：
- 二元结果（是/否）
- 动态赔率（根据供需调整）
- 全额赔付（猜对拿 $1/份额）
- 随时退出（二级市场交易份额）

## 📦 快速开始

### 环境要求

- Node.js 18+
- npm / yarn / pnpm
- OKX API 密钥

### 安装

```bash
# 1. 克隆项目
git clone https://github.com/yourname/onchain-casino.git
cd onchain-casino

# 2. 安装依赖
pnpm install

# 3. 配置环境变量
cp .env.example .env.local
# 编辑 .env.local 填入你的配置

# 4. 启动开发服务器
pnpm dev

# 5. 访问 http://localhost:3000
```

### 环境变量

```bash
# OKX API
OKX_API_KEY=your_api_key
OKX_SECRET_KEY=your_secret_key
OKX_PASSPHRASE=your_passphrase

# 部署
NEXT_PUBLIC_APP_URL=https://your-app.vercel.app
```

## 🎮 玩法介绍

### 新手入门流程

```
1. 连接钱包 (Solana Phantom / EVM MetaMask)
   ↓
2. 存入 USDT (作为游戏资金)
   ↓
3. 选择玩法：
   ├─ 预测市场 → 选择预测题目 → 下注 → 等待结果
   ├─ 组队打新 → 创建/加入小队 → 投票 → 自动买入
   ├─ 对赌协议 → 发起/接受挑战 → 锁定资金 → 自动结算
   ├─ 大逃杀 → 支付入场费 → 交易 → 排名分奖金
   └─ 带单分润 → 选择带单人 → 跟投 → 自动分润
   ↓
4. 赚取收益 + 解锁成就
```

### 高级技巧

| 技巧 | 说明 | 风险 |
|------|------|------|
| 对冲下注 | 同时买 Yes/No，锁定利润 | 低 |
| 跟单策略 | 跟随高胜率玩家 | 中 |
| 分散投资 | 参与多个预测市场 | 低 |
| 及时止损 | 设置自动止损 | 中 |
| 逆向思维 | 跟"韭菜集中营"反向 | 高 |
